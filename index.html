宇多田光

宇多田光（Utada Hikaru），1983年1月19日出生于美国纽约州曼哈顿，美籍日裔女歌手、词曲作者、音乐制作人以及电台主持，昵称Hikki（ヒッキー）

重要事件

  • 1999年，发行首张日文专辑《First Love》，创下日本及亚洲专辑销量纪录 。
  • 2001年，发行第2张日文专辑《Distance》，创下日本专辑首周销量纪录 ，并与《First Love》成为日本专辑销量前二的原创专辑。
  • 2002年，发行第3张日文专辑《Deep River》 。
  • 2004年，以“Utada”名义发行首张英文专辑《Exodus》 ，同年凭借首张精选辑《Utada Hikaru SINGLE COLLECTION VOL.1》第4次获得日本年度专辑销量冠军 。
  • 2006年，发行第4张专辑《ULTRA BLUE》 。
  • 2007年，为电视剧《花样男子2》演唱主题曲《Flavor Of Life》 。200年，推出第5张专辑《HEART STATION》 。
  • 2010年，宣布退出歌坛 ，并发行第2张精选辑《Utada Hikaru SINGLECOLLECTION VOL.2》 。
  • 2016年，正式重返乐坛，并发行第6张日文专辑《幻影》 ，该专辑获得第5届日本唱片大奖“最优秀专辑奖” 。
  • 2018年，发行第7张日文专辑《初恋》 ，同年举办“Hikaru UtadaLaughter in the Dark Tour 2018”巡回演唱会 。
  • 2021年，发行首张迷你专辑《One Last Kiss》 。

个人成就

作为唱作人，宇多田光包揽大部分作品的作词与作曲，并在2004年开始独立编曲。截至2010年，宇多田光在日本的唱片总销量超过3800万张 。

生活照

人物评价

1998年12月在日本的电视和广播里登场后，宇多田光在90年代后半期至2000年代中期成为了日本流行乐坛的天后，自出道曲《Automatic/time wiiltell》开始不断打破着乐坛的记录。她以日式R&B席卷乐坛，又不拘泥于类型音乐的限制尝试不同风格的创作；在日本乃至亚洲取得了成功后，继以开拓和谦虚进军美国乐坛；她不仅是毫无异议的日本音乐的代表性歌手，亦是一位世界性的音乐家。她的歌曲中，没有枯燥乏味的陈年老调，而是在不断的成长和丰富，演绎着属于自己独一无二的人生和音乐，依旧闪烁着光辉。她那种存有悲伤深刻的脆弱精致的声音，能激发听众的保护本能，是给听者带来力量的好声（BARKS、松任谷由实评）。

无论是销量，全球影响力，还是对日本乐坛的贡献与革新，宇多田光都是平成年代独一份的存在，是没有一个日本歌手能够与其相提并论的。特别是宇多田光独特的音乐风格、唱法上的革新、以及在音乐形式上的不断探索，是宇多田光吸引力不减的重要原因 。

因为出生于美国，宇多田光在唱法上深受欧美唱腔的影响，其标准的英语发音也是区别于其他日本歌手的存在。利用这一独特优势，宇多田光在出道时就实现了对于日语歌曲在唱法上的突破与创新。简单来说，早期出道那段时间，宇多田光是把日语的罗马音拆分开当英语来唱，并且将这种唱法与R&B结合起来，成功地将这种音乐形式本土化。仅这一点可以奠定宇多田光在整个日本乐坛的地位。与此同时，宇多田光的音乐在旋律上的运用也非常大胆，编曲中也会运用传统日式编曲甚至是演歌等日本传统音乐，这也是在20世纪90年代的日本乐坛中非常特殊的存在。从宇多田光的创作水平来看，旋律绕耳，专辑内歌曲流行性大于实验性，但绝对不落俗套，不走模式套路，她才华尽显，创作的歌曲好听易传唱。从整个亚洲音乐区来看，引入R&B的歌手并不只有宇多田光，但由她造成的影响力无疑是惊人且深远的。

随着年龄的增长以及阅历的增加，宇多田光也在音乐形式和唱法上不断突破，不论是作词还是作曲都有了更多思考。由于自己在演唱上气息不足的短板比较明显，宇多田光一直在探索更加适合自己的唱法，并对现有流行乐坛的唱法进行革新，而不是一味追求主流音乐审美，从这一点上来说，她要比滨崎步等同期歌手更聪明，也更能表现出她作为一个音乐人在艺术上的追求。

自2012年11月发布的《樱花流》后，暌违三年半的回归之作《将花束献给你》和《盛夏阵雨》一同发布。两首歌出来以后，很多人评价称再次被宇多田光治愈，歌里有寂寞的影子，却又非常温柔、坚实，是宇多田光更为成熟的面貌，着实令人感动。歌姬混战的时代早已终结，在唱片业持续式微、媒体急速膨胀之时，流行音乐的发展到了技术革新和能量无以守恒的紧张阶段，乐坛整体颓势明显、大物不再。宇多田光的回归，让人们依旧愿意爱着那些感动着、深刻影响着自己的歌手，依旧会把美好的旋律植根于脑海和记忆，也依旧渴望能留住值得回味的声音和歌曲。
